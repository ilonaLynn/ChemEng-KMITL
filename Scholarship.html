เงื่อนไขการรับทุน

ทุนละ 5,000 บาทต่อปีการศึกษา งานวิชาการนักศึกษาผู้ขอรับทุนเป็นผู้เสนอชื่อต่อคณะกรรมการกองทุนการศึกษา



ทุนผู้สร้างชื่อเสียงในนามสถาบัน

ด้านการประกวด/การแข่งขันวิชาการ

• ระดับโลก : หลายทวีป ผู้เข้าแข่งขันไม่น้อยกว่า 16 ประเทศ อันดับ 1 => 100,000 บาท อันดับ 2 => 50,000 บาท อันดับ 3 => 25,000 บาท

• ระดับนานาชาติ : ผู้เข้าแข่งขันไม่น้อยกว่า 8 ประเทศ อันดับ 1 => 50,000 บาท อันดับ 2 => 25,000 บาท อันดับ 3 => 13,000 บาท

• ระดับเอเชีย : นอกเหนือจากอาเซียน ผู้เข้าแข่งขันไม่น้อยกว่า 5 ประเทศ อันดับ 1 => 40,000 บาท อันดับ 2 => 20,000 บาท อันดับ 3 => 10,000 บาท

• ระดับอาเซียน : ผู้เข้าแข่งขันไม่น้อยกว่า 5 ประเทศ อันดับ 1 => 25,000 บาท อันดับ 2 => 13,000 บาท อันดับ 3 => 7,000 บาท

• ระดับประเทศ : ผู้เข้าแข่งขันไม่น้อยกว่า 16 มหาวิทยาลัย อันดับ 1 => 20,000 บาท อันดับ 2 => 10,000 บาท อันดับ 3 => 5,000 บาท

การแข่งขันทางวิชาการระดับประเทศที่ไม่เข้าเกณฑ์ หากคณะกรรมการเห็นว่ามีการแข่งขันสม่ำเสมอทุกปีและจัดโดยหน่วยงานที่มีชื่อเสียง อาจให้เพิ่มการแข่งขัน เพื่อให้ทุนแก่นักศึกษาเป็นกรณีพิเศษ งานวิชาการนักศึกษาเสนอชื่อหน่วยงานที่จัดแข่งขัน ให้คณะกรรมการกองทุนการศึกษาพิจารณา

ด้านการกีฬา

• กีฬาโอลิมปิกระดับโลก : หลายทวีปรวมกัน ผู้เข้าแข่งขันไม่น้อยกว่า 16 ประเทศ เหรียญทอง => 100,000 บาท เหรียญเงิน => 50,000 บาท เหรียญทองแดง => 25,000 บาท

• กีฬามหาวิทยาลัยโลก : หลายประเทศผู้เข้าแข่งขันไม่น้อยกว่า 8 มหาวิทยาลัย เหรียญทอง => 60,000 บาท เหรียญเงิน => 30,000 บาท เหรียญทองแดง => 15,000 บาท

• กีฬาระดับเอเชีย : นอกเหนือจากประเทศกลุ่มอาเซียนผู้เข้าแข่งขันไม่น้อยกว่า 5 ประเทศ เหรียญทอง => 50,000 บาท เหรียญเงิน => 25,000 บาท เหรียญทองแดง => 13,000 บาท

• กีฬาระดับอาเซียน : กลุ่มอาเซียนผู้เข้าแข่งขันไม่น้อยกว่า 5 ประเทศ เหรียญทอง => 30,000 บาท เหรียญเงิน => 15,000 บาท เหรียญทองแดง => 8,000 บาท

• กีฬามหาวิทยาลัยแห่งประเทศไทย : ผู้เข้าร่วมแข่งขันไม่น้อยกว่า 16 มหาวิทยาลัยระดับประเทศ อันดับ 1 => 20,000 บาท อันดับ 2 => 10,000 บาท อันดับ 3 => 5,000 บาท

ด้านอื่นๆ

งานวิชาการของนักศึกษาผู้ที่ได้รับรางวัล เป็นผู้เสนอขอต่อคณะกรรมการกองทุนการศึกษาเพื่อพิจารณา

• ระดับประเทศ ยกเว้นค่าธรรมเนียมการศึกษา 2 ภาคการศึกษา นับถัดจากภาคการศึกษาที่ได้รับการอนุมัติทุนการศึกษา

• ระดับนานาชาติขึ้นไป ยกเว้นค่าธรรมเนียมการศึกษาตลอดหลักสูตร นับถัดจากภาคการศึกษาที่ได้รับอนุมัติทุนการศึกษา

หมายเหตุ เงื่อนไขการรับทุน แบ่งเป็น 2 กรณี • กรณีประเภทบุคคล : รับทุนเป็นรายบุคคล • กรณีประเภททีม : รับทุนเป็นรายทีม โดยได้รับทุนเป็น 2 เท่าของเงินทุน และให้หารเฉลี่ยทุกคนซึ่งได้รับทุนเฉพาะนักศึกษา

นักศึกษาชั้นปีที่ 1-4 ที่ทำประโยชน์ให้แก่สถาบันด้านต่างๆ เช่น ด้านกิจกรรม ด้านจิตอาสา ด้านวิชาการ มีความประพฤติเรียบร้อย

เงื่อนไขการรับทุน

ทุนละ 5,000 บาท งานวิชาการของนักศึกษาผู้ทำคุณประโยชน์เป็นผู้เสนอชื่อต่อคณะกรรมการกองทุนการศึกษา



ทุนสนับสนุนการนำเสนอผลงานทางวิชาการ

คุณสมบัติ

ไม่เป็นผู้ได้รับทุนการศึกษาเต็มรูปแบบจากแหล่งทุนอื่นที่มีการสนับสนุนค่าใช้จ่ายในการตีพิมพ์ผลงานทางวิชาการไว้แล้วนักศึกษามีสิทธิ์เบิกได้ทางใดทางหนึ่งเท่านั้น และไม่เป็นผู้ได้รับยกเว้นค่าธรรมเนียมการศึกษาจากสถาบัน

หลักเกณฑ์การสนับสนุน

• หลักเกณฑ์การสนับสนุน
1. การเผยแพร่ผลงานที่จะได้รับทุนสนับสนุนการนำเสนอผลงานวิชาการ การเผยแพร่ผลงานในวารสารวิชาการที่มีมาตรฐาน การนำเสนอผลงานทางวิชาการในที่ประชุมวิชาการรูปแบบบทความวิจัยมีกรรมการมาร่วมกลั่นกรองเป็นที่ยอมรับในวงการวิชาชีพสาขานั้นๆ วารสารวิชาการที่นักศึกษาลงตีพิมพ์ต้องได้รับความเห็นชอบจากคณะกรรมการประจำส่วนงานวิชาการ ระดับบัณฑิตศึกษาใช้เงื่อนไขในการสำเร็จการศึกษาได้
2. ผลงานที่นักศึกษาขอรับทุนต้องเป็นส่วนหนึ่งของการศึกษาในหลักสูตรที่ตนกำลังศึกษาอยู่
3. นักศึกษาต้องมีชื่อเป็นเจ้าของผลงานร่วมกับชื่ออาจารย์ที่ปรึกษา

• หลักเกณฑ์การจ่ายทุนสนับสนุน
1. นักศึกษาต้องแสดงหลักฐานการลงทะเบียนเข้าร่วมประชุมวิชาการโดยระบุชื่อนักศึกษาที่เป็นผู้ลงทะเบียนเท่านั้น
2. จำนวนครั้งที่มีสิทธิ์ได้รับทุนสนับสนุน ปริญญาตรี => 1 ครั้งตลอดหลักสูตรการศึกษา ปริญญาโท => ไม่เกิน 1 ครั้งต่อปีหรือ 2 ครั้งตลอดหลักสูตรการศึกษา ปริญญาเอก => ไม่เกิน 1 ครั้งต่อปีหรือ 3 ครั้งตลอดหลักสูตรการศึกษา และในกรณีที่ไม่สามารถใช้เป็นเงื่อนไขในการสำเร็จการศึกษาได้ได้รับทุนสนับสนุนไม่เกิน 1 ครั้งตลอดหลักสูตรนักศึกษา
3. อัตราการจ่ายเงินทุนสนับสนุนการนำเสนอผลงานวิชาการ 5,000 บาทต่อบทความที่ได้ลงตีพิมพ์ในวารสารวิชาการมีใบเสร็จรับเงินค่าธรรมเนียมการตีพิมพ์ การประชุมวิชาการในประเทศระดับชาติไม่เกิน 4,500 บาท ในประเทศระดับนานาชาติไม่เกิน 7,500 บาท (ค่าลงทะเบียนในการเข้าร่วมประชุมวิชาการ, ค่าเดินทางไปและกลับตามที่จ่ายจริง, ค่าที่พักตามที่จ่ายจริงแต่ไม่เกินอัตราคนละ 1,000 บาทต่อคืน) การเผยแพร่ผลงานในต่างประเทศของนักศึกษาไม่เกิน 12,000 บาท (ค่าลงทะเบียนในการเข้าร่วมประชุมวิชาการ, ค่าเดินทางไปและกลับตามที่จ่ายจริง, ค่าที่พักตามที่จ่ายจริง) ต้องดำเนินการยื่นขออนุมัติรับทุนต่อรองอธิการบดีฝ่ายพัฒนานักศึกษาและศิษย์เก่าสัมพันธ์ โดยยื่นก่อนวันส่งผลงานไปเผยแพร่อย่างน้อย 7 วัน



ทุนสนับสนุนการแลกเปลี่ยนและฝึกงานต่างประเทศ

คุณสมบัติ

กำลังศึกษาระดับปริญญาตรีเกรดเฉลี่ยสะสมไม่น้อยกว่า 3.00 และไม่เป็นผู้ทำผิดกฎหมายหรือวินัยนักศึกษา

หลักเกณฑ์การขอรับทุน

• หลักเกณฑ์การขอรับทุนสนับสนุน
1. การฝึกงานต่างประเทศต้องดำเนินการร่วมกับสถาบันที่อยู่ใน 300 อันดับแรกของการจัดอันดับมหาวิทยาลัยโลกของปีการศึกษาที่ขอรับทุน หรือสถานประกอบการที่สถาบันมีข้อตกลงความร่วมมือกัน
2. นักศึกษาผู้ขอรับทุนต้องติดต่ออาจารย์ที่ปรึกษา และสถาบันในต่างประเทศให้ตอบรับเป็นนักศึกษาแลกเปลี่ยน
3. นักศึกษาผู้ขอรับทุนต้องนำหลักฐานการตอบรับมายื่นต่อคณะอนุกรรมการพิจารณาทุนและเปลี่ยนนักศึกษาเพื่อพิจารณาคัดเลือก
4. กรณีนักศึกษาที่ได้รับทุนสนับสนุนการแลกเปลี่ยนแล้ว แต่ยังไม่ได้รับเต็มจำนวนค่าใช้จ่ายทั้งหมด ส่วนงานวิชาการสามารถสนับสนุนทุนในลักษณะนี้ให้กับนักศึกษาได้ แต่ต้องไม่เกินค่าใช้จ่ายที่เกิดขึ้นจริง
5. นักศึกษาผู้ขอรับทุนต้องทำหนังสือยินยอมรับทุนต่อสถาบัน ก่อนจะเดินทางไปแลกเปลี่ยน
6. ระยะเวลาการแลกเปลี่ยนแบ่งออกเป็น
6.1 สถานประกอบการต่างประเทศที่มีบันทึกข้อตกลงความร่วมมือกับสถาบัน (ระยะเวลาแลกเปลี่ยนน้อยกว่า 30 วัน, ระยะเวลาแลกเปลี่ยนอย่างน้อย 30 วันขึ้นไป)
6.2 สถานประกอบการต่างประเทศที่ไม่มีบันทึกข้อตกลงความร่วมมือกับสถาบันต้องมีระยะเวลาแลกเปลี่ยนอย่างน้อย 90 วันขึ้นไป

• หลักเกณฑ์การจ่ายเงินสนับสนุน
1. เป็นผู้รับทุนตามหลักเกณฑ์ข้อ 6.1 และไม่มีทุนอื่นสนับสนุน รับทุนแบบเหมาจ่าย 50,000 บาท
2. เป็นผู้รับทุนตามหลักเกณฑ์ข้อ 6.1 หรือ 6.2 รับทุนแบบเหมาจ่ายตามโซนประเทศ
สหรัฐอเมริกา และแคนาดา => 60,000 บาท
ประเทศในยุโรป ออสเตรเลีย และนิวซีแลนด์ => 50,000 บาท
ญี่ปุ่น สิงคโปร์ เกาหลีใต้ จีน ไต้หวัน และฮ่องกง => 40,000 บาท
ประเทศอื่นๆ นอกเหนือที่กำหนด => 20,000 บาท

• รายงานผลการไปแลกเปลี่ยนหรือฝึกงานต่างประเทศ
1. นักศึกษาผู้รับทุนต้องรายงานผลการเข้าร่วมกิจกรรมการแลกเปลี่ยนทางวิชาการต่อคณะอนุกรรมการพิจารณาทุนแลกเปลี่ยนนักศึกษาภายใน 30 วันนับตั้งแต่วันถัดจากวันกลับมายังประเทศไทย
2. หากไม่รายงานผลการเข้าร่วมกิจกรรมภายในกำหนดระยะเวลาจะไม่ได้รับทุนการสนับสนุนในครั้งถัดไป

เงื่อนไขการชดใช้คืน

1. กิจกรรมการแลกเปลี่ยนหรือฝึกงานถูกยกเลิกโดยสถาบันอุดมศึกษาต่างประเทศ
2. นักศึกษาจำเป็นต้องยุติกิจกรรมการแลกเปลี่ยนหรือฝึกงานลง แต่ไม่ครบกำหนดระยะเวลา คือ เดินทางกลับก่อนครบกำหนด 30 วันหรือ 90 วัน แล้วแต่กรณี
3. สถาบันพบว่านักศึกษาไม่ได้เข้าร่วมกิจกรรมการแลกเปลี่ยนหรือฝึกงานจริงตามที่ระบุไว้ตอนสมัครรับทุน
4. นักศึกษากระทำความผิดระหว่างเข้าแลกเปลี่ยนหรือฝึกงาน สถาบันเสื่อมเสียชื่อเสียง ให้คณะอนุกรรมการพิจารณาทุนแลกเปลี่ยนนักศึกษามีอำนาจเรียกตัวนักศึกษากลับและขอเรียกทุนสนับสนุนคืน
5. กรณีตามข้อ 1 และข้อ 2 คณะอนุกรรมการพิจารณาทุนแลกเปลี่ยนนักศึกษาอาจพิจารณายกเว้นการชดใช้คืนเงินทุนได้
6. หากนักศึกษาผู้รับทุน จำเป็นต้องมีการเปลี่ยนสถาบันอุดมศึกษาในต่างประเทศ จากที่เสนอตอนสมัครขอรับทุนนักศึกษา ต้องได้รับความเห็นชอบจากคณะอนุกรรมการพิจารณาทุนแลกเปลี่ยนนักศึกษาก่อน
